3D-DART Gallery
This gallery shows you the capabilities of the 3D-DART server in creating DNA structural models with a defined conformation. These are real life examples in which a model is generated based on a DNA structure in complex with a protein. This shows that the server is able to recreate the global conformation of the DNA structure in the various complexes.
You can view all models using the embedded Jmol viewer. Alternatively you can also download the server output
All structures of the complex are in green and the models in blue
